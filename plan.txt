####### Data Driven Test ###############

Welcome to day 21 of our Playwright plus JS. 
Today we are going to be talking about data-driven testing. 

| Concept                    | Meaning                                                                                |
| -------------------------- | -------------------------------------------------------------------------------------- |
| **Data-driven test**       | Running the same workflow multiple times with different inputs.                        |
| **Why it matters**         | Validates multiple scenarios quickly, keeps tests DRY.                                 |
| **Where data lives**       | In `test-data/` files (JSON / JS objects).                                             |
| **What makes it scalable** | The Page Object Model + loop structure + fixtures = clear separation of data vs logic. |
